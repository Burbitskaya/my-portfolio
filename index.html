Projects

Geoportal

This project is a full-featured mapping web application that allows users to create, view, and manage various types of layers, build custom maps, and save or export them. It includes tools for measuring distances and areas, adding notes to maps, and searching locations by address. The application is built with Next.js and ArcGIS Maps SDK for JavaScript, with user, resource, and map data stored in MongoDB. Authentication is handled via NextAuth.js, and the user interface is designed with the Calcite Design System, while tables are efficiently managed using MUI Grid.
